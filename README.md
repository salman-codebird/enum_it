# enum_it

Automate DNS enumeration

  ->find AA, CNAME, MX, TXT and AAAA records of a domain
  ->find all subdomains [using sublist3r]
  ->find all alive sub domains
  ->change domain names and keep repeating

Dependencies
  ->sublist3r
  ->httprobe

To run
  ->cd enum_it
  ->chmod +x enum_it
  ->./enum_it
